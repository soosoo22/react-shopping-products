    - electronics

    - fitness
    - books
  - [x] 상품을 가격 순으로 정렬한다.
    - 낮은 가격 순 (asc)
    - 높은 가격 순 (desc)
  - [x] 장바구니에 상품 담기

    - [x] 장바구니에 담긴 아이템 종류의 갯수로 숫자를 표시한다.
